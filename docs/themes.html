Créer votre propre thème

Vous pouvez créer vos propres thèmes en définissant des variables CSS personnalisées :

/* Définir un thème personnalisé */ [data-theme="mon-theme"] { --primary-color: #ff6b6b; --secondary-color: #4ecdc4; --success-color: #45b7d1; --danger-color: #e74c3c; --warning-color: #f9ca24; --info-color: #3498db; --bg-primary: #f8f9fa; --text-primary: #2c3e50; --border-radius: 12px; --shadow: 0 4px 20px rgba(0, 0, 0, 0.1); } /* Utilisation */ <html data-theme="mon-theme"> <!-- Votre contenu --> </html>
